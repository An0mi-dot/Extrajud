Atualizações

Este painel permite configurar um endpoint JSON (um arquivo updates.json) que descreve a versão mais recente do aplicativo. Use sempre a URL direta para o arquivo JSON (por exemplo o link raw do GitHub). Exemplo de campos esperados no JSON: version, url (link para artefato), raw, notes e published_at. Ao clicar em Verificar agora o app buscará esse JSON e comparará com a versão instalada; se houver atualização ela será exibida com changelog e link. Se Auto-Update estiver habilitado, o link de download será aberto automaticamente. Atenção: se você apontar para uma página HTML (ex.: uma página de Release no GitHub) em vez do arquivo raw, a verificação falhará com uma mensagem de erro e — quando possível — o app sugerirá o link raw correto. Para conveniência, você pode usar o arquivo de atualização hospedado neste repositório:
https://raw.githubusercontent.com/An0mi-dot/Extrajud/main/updates.json

Sobre a Ferramenta

O EXTRATJUD é uma solução corporativa de automação robótica de processos (RPA) desenvolvida para otimizar as operações do setor jurídico da Neoenergia Coelba.

Nota Técnica: O aplicativo não utiliza APIs oficiais dos tribunais, devido à indisponibilidade de acesso público a esses recursos. Em vez disso, a ferramenta automatiza processos manuais, utilizando Inteligência Artificial para executar a navegação e as tarefas repetitivas da forma mais rápida possível.

A ferramenta emprega tecnologias de navegação headless para interagir com portais judiciais de forma segura e eficiente, permitindo a extração massiva de dados, geração de relatórios gerenciais e auditoria com evidências documentais.

Ficha Técnica

  • Engine: Electron & Node.js
  • Automação: Microsoft Playwright
  • Interface: HTML5 / CSS3 (Custom Theme)
